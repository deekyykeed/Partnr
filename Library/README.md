## How to add something

**Transcript:** Drop the `.md` file in `Transcripts/` — name it clearly e.g. `alex-hormozi-offer-creation.md`

**Book:** Create a file in `Books/` with the title as the filename. Include: summary, key quotes, and how to apply it.

**Article:** Save to `Articles/` with source URL at the top.

**Framework:** Save to `Frameworks/` with a clear explanation of the model and how to use it in the business.

Then update the table above so everything stays findable.
